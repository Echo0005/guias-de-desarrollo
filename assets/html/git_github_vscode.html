Github, Git y Visual Studio Code: Clonado y subir Commits desde VS Code.

Clonado:

El Clonado consiste en traernos/descargarnos un Repositorio Remoto existente. Trabaja a partir del comando git clone seguido de una URL completa.

Paso a paso:
Usaremos de ejemplo para la URL el repositorio en Github de las pruebas iniciales del proyecto InterLinked: https://github.com/kevinfonttlizama/Interlinked-base

  • Abrir localmente Git por Consola/Terminal, en la ubicacion que prefiramos.
  • Utilizaremos el siguiente comando: git clone https://github.com/kevinfonttlizama/Interlinked-base
  • Y ya tendremos nuestro Repositorio Remoto clonado.

Commits desde VS Code:

En nuestra Barra Principal lateral (izquierda, dispuesta verticalmente) disponemos del Control de código fuente:

Cuando hagamos una modificacion (y guardemos), podemos proseguir con el Commit y Push:

  • Al hacer cambios se habilita el boton Confirmar.
  • En el Input, arriba de este boton Confirmar, podemos colocar el Mensaje que tendra nuestro Commit.
  • En el boton Confirmar, click en su menu desplegable y seleccionamos Hacer "commit" e incertar.
