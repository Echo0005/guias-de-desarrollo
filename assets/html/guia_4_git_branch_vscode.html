Notas

Las Ramas mas actualizadas apareceran mas arriba en la lista de branches de GitLens. Nos permite tener conciencia del estado de cada rama. La ultima rama en la que trabajemos queda arriba, al volver a Rama main y realizar un merge, main quedara arriba de todas.
