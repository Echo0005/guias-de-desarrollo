Github, Git y Visual Studio Code: Clonado y subir Commits desde VS Code

Esta guia proporcionara una orientacion rapida para poder Clonar un Repositorio Remoto y realizar un Push forzado. Respecto a este Push Forzado es la manera mas rapida, pero no recomendado, de subir nuestras modificaciones al Repositorio Remoto.

A su vez, esta guia debera tomarselo como una introduccion muy sencilla, a trabajar con Github y Git.

Clonado

El Clonado consiste en traernos/descargarnos un Repositorio Remoto existente. Trabaja a partir del comando git clone seguido de una URL completa.

Paso a paso:

Usaremos de ejemplo para la URL, el repositorio en Github de las pruebas iniciales del proyecto InterLinked:

Link: https://github.com/kevinfonttlizama/Interlinked-base

  • Abrir localmente Git por Consola/Terminal, en la ubicacion que prefiramos.
  • Utilizaremos el siguiente comando: git clone https://github.com/kevinfonttlizama/Interlinked-base

Repositorio Remoto clonado.

Commit y Push (subir al Repositorio Remoto) realizados.
